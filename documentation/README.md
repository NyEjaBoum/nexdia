# Nexdia Nuxt Project

Ce projet est une application web construite avec Nuxt.js (framework Vue 3) pour la plateforme Nexdia.

## Structure du projet

- **assets/** : Fichiers statiques comme les feuilles de style CSS.
- **components/** : Composants Vue réutilisables (navbar, footer, sections, cartes, etc.).
- **data/** : Données statiques et thèmes utilisés dans l'application.
- **layouts/** : Layouts globaux pour l'application.
- **pages/** : Pages principales du site (accueil, packs, contact, FAQ, etc.).
- **packs/** : Pages dynamiques pour chaque pack.

## Prérequis

- Node.js (version recommandée : 18+)
- npm ou yarn

## Installation

1. Ouvrez un terminal dans le dossier `nexdia-nuxt`.
2. Installez les dépendances :

```bash
npm install
```

ou

```bash
yarn install
```

## Lancer le projet en développement

```bash
npm run dev
```

ou

```bash
yarn dev
```

Le site sera accessible sur `http://localhost:3000`.

## À propos

Ce projet a été conçu pour présenter la plateforme Nexdia, ses packs, témoignages, FAQ, et permettre le contact avec la communauté. L'architecture est modulaire pour faciliter l'ajout de nouvelles fonctionnalités.

Pour toute question, consultez les fichiers dans le dossier `components/` et `pages/` pour comprendre la structure et le fonctionnement de l'application.
